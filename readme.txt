Show and manage SSL secure seal or custom image on WooCommerce pages.

== Description ==

If you are accepting credit card payment, it is necessary to have a SSL Certificate installed. Adding a secure seal badge was a next step to bring confidence for your customers and could increase conversions. When it placed close to the payment button, your customer knows that the site where they entering their card information on is 100% secure.

This plugin lets you easily show and manage SSL secure seal or custom image on certain WooCommerce pages, most importantly the checkout page. 

Features:
 - Able to upload or select image from WordPress media as the seal badge.
 - Or, put the SSL Certificate seal embed code directly.
 - Provided with default badge for your convenience

= Where to Find Settings Screen =
  
  WooCommerce > Settings > SSL Seal Tab

<strong>Easily change and manage which pages you want to display the secure seal badge.</strong>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/ssl-secure-seal-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce->Settings->Seal screen to configure the plugin
1. Select or Upload custom image for the seal or just enter the seal embed code in the appropriate setting. Save the changes.
1. No more steps. Show the seal on each WooCommerce pages so customers have confidence and trust into your e-commerce site.


== Frequently Asked Questions ==

= Where to Find Settings Screen =

  WooCommerce > Settings > Seal Tab
  
= Do I need to do anything after plugin is activated =

Yes, please visit the Setting screen, select appropriate WooCommerce pages you want the seal showing up.
